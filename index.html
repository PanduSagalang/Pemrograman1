HTML (HyperText Markup Language) diciptakan oleh Tim Berners-Lee pada tahun 1991. HTML merupakan bahasa markup yang digunakan untuk membuat halaman web. Awal mula HTML HTML awalnya diciptakan untuk memudahkan pengelolaan dokumen HTML versi pertama dirilis pada tahun 1993.
HTML dirancang untuk mendukung teks dan hyperlink sederhana HTML digunakan terbatas di kalangan para ilmuwan untuk mengakses dokumen Perkembangan HTML HTML 2.0 diterbitkan pada tahun 1995, yang berisi semua fitur HTML 1.0 bersama dengan beberapa fitur tambahan HTML 4.01 merupakan versi sukses dari HTML sebelum HTML 5.0 HTML 5 dirilis pada tahun 2014 dan bisa menggabungkan HTML dengan bahasa pemrograman lainnya seperti CSS dan JavaScript Standar HTML World Wide Web Consortium (W3C) hadir sebagai lembaga yang salah satu fungsinya adalah menjaga patokan baku dari bahasa HTML Standar yang diikuti untuk membuat sebuah file HTML mengacu pada ASCII (American Standard Code for Information Interchange)
